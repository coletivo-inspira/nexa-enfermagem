# Nexa Enfermagem - Brandbook & Diretrizes da Marca

**Nexa Enfermagem** é especializada em **Atendimento Domiciliar**, levando cuidado que vai até você.

---

## 🎨 Identidade Visual e Valores

A identidade da Nexa foi construída com base nos pilares de **confiança, cuidado e tranquilidade**, refletindo a essência da enfermagem e o profissionalismo em cada atendimento.

### Elemento Símbolo: A Lâmpada da Enfermagem
O logotipo apresenta o símbolo clássico da **Lâmpada da Enfermagem** (símbolo internacional da profissão, remetendo a Florence Nightingale) de forma estilizada e moderna, dentro de um contorno hexagonal. Esse símbolo representa a luz, o cuidado, a vigilância contínua e a dedicação aos pacientes, sendo o núcleo central da marca.

### Paleta de Cores
As cores foram escolheras para transmitir segurança, calma e saúde:

*   **Verde Escuro Principal:** `#0D7C74`
*   **Verde Água (Destaque):** `#6CCFC9`
*   **Verde Gelo (Fundo/Apoio):** `#E6F5F3`
*   **Cinza Escuro (Textos):** `#333333`

### Tipografia
As tipografias escolhidas visam transmitir **modernidade, leveza e profissionalismo**:

*   **Montserrat** (Indicada para títulos e informações de impacto)
*   **Poppins** (Indicada para subtítulos e corpo de texto)

A essência da comunicação da Nexa Enfermagem foca em acolhimento e qualidade. Nossos pilares de comunicação são:

## 💻 Estrutura Técnica da Landing Page

A Landing Page foi estruturada como um projeto web moderno e de alta performance.

### Stack Tecnológica
- **React 19 & Vite 8:** Aplicação web SPA rápida e responsiva.
- **Vanilla CSS3:** Estilização com CSS Custom Properties (Variáveis) baseada no Brandbook.
- **Lucide React:** Biblioteca moderna de ícones vetoriais.

### Arquitetura de Pastas
```text
nexa-enfermagem/
├── public/                 # Imagens e ativos visuais da marca
├── src/
│   ├── components/         # Componentes React modularizados
│   │   ├── Header.jsx      # Menu e navegação responsiva
│   │   ├── Hero.jsx        # Banner de introdução e chamadas para ação
│   │   ├── Services.jsx    # Seção com cards interativos dos serviços
│   │   ├── About.jsx       # Perfil profissional com competências e histórico
│   │   ├── Manifesto.jsx   # Depoimentos, manifesto e valores centrais
│   │   ├── WhatsAppFloat.jsx # Botão flutuante para contato direto
│   │   └── Footer.jsx      # Rodapé corporativo com informações técnicas
│   ├── App.jsx             # Ponto de montagem das seções
│   ├── index.css           # Variáveis de design e reset global
│   └── main.jsx            # Ponto de entrada da aplicação React
├── index.html              # HTML base configurado para SEO
├── package.json            # Dependências e scripts do npm
└── vite.config.js          # Configurações de build do Vite
```

### Como Executar Localmente

1. **Instalar Dependências:**
   ```bash
   npm install
   ```

2. **Iniciar Servidor de Desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Gerar Build de Produção:**
   ```bash
   npm run build
   ```
